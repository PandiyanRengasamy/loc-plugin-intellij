  Contains: "CodeEventRequest"

  Contains: linesAdded=N (your line count)
  Contains: genAiTool=GitHub Copilot

  Data: Contains your event

IF ALL ✅: TEST PASSING

## 🔧 Setting Up Test Environment

### Prerequisites
```
✅ IntelliJ IDEA 2025.1+
✅ Java 21+
✅ GitHub Copilot plugin installed
✅ LOC Plugin built
✅ Test project created
```

### Configuration

**Settings → Tools → GenAI LOC Tracker**
```
Enabled:            ✅ Checked
Backend URL:        http://localhost:8080/api/events
Batch Size:         10
Flush Interval:     30 seconds
```

### Optional: Backend Setup
```
If testing with backend:
  Start your backend service on http://localhost:8080

If testing offline:
  Set URL to unreachable: http://localhost:9999
  OR stop backend service
  CSV will be created automatically
```
